### Installation
requires Gulp 


__1. Get the repository and install the dependencies and devDependencies__
  
  ```sh
  $ git clone https://github.com/Skrij/hetic-p2019-14
  $ cd hetic-p2019-14
  $ git checkout dev
  $ npm install -d
  ```
  
__2. Start the project.__
  
  ```sh
  $ gulp build 
  ```

__3. Open /dist in browser.__
